[Bun](https://bun.sh/) es una herramienta todo en uno que combina un entorno de ejecución de JavaScript y una serie de utilidades. Consulta la [documentación de Bun](https://bun.com/docs) para más información.

:::caution
Al usar Bun con Astro pueden surgir algunos inconvenientes. Es posible que algunas integraciones no funcionen como se esperaba. Consulta [la documentación oficial de Bun para trabajar con Astro](https://bun.com/guides/ecosystem/astro) para más detalles.

Si tienes algún problema al utilizar Bun, por favor, [abre una incidencia en GitHub directamente con Bun](https://github.com/oven-sh/bun/issues/new/choose).
:::

## Prerrequisitos

- Bun instalado localmente en tu máquina. Consulta las [instrucciones de instalación](https://bun.com/docs/installation) en la documentación oficial de Bun.

## Crea un nuevo proyecto Astro con Bun

Crea un nuevo proyecto Astro con Bun utilizando el siguiente comando `create-astro`:

```bash
bun create astro mi-proyecto-astro-usando-bun
```

## Instalar dependencias

Si te has saltado el paso "¿Instalar dependencias?" durante el asistente de la CLI, asegúrate de instalar las dependencias antes de continuar.

```bash
bun install
```

## Agregar tipos

Bun publica el paquete [`@types/bun`](https://www.npmjs.com/package/@types/bun), que contiene los tipos en tiempo de ejecución de Bun.

Instala `@types/bun` con el siguiente comando:

```sh
bun add -d @types/bun
```

## Opciones de instalación de la CLI

### Uso de integraciones

También puedes utilizar cualquiera de las integraciones oficiales de Astro con el comando `astro add`:

```bash
bun astro add react
```

### Usa un tema o una plantilla inicial

Puedes iniciar un nuevo proyecto Astro basado en un [ejemplo oficial](https://github.com/withastro/astro/tree/main/examples) o en la rama principal de cualquier repositorio de GitHub pasando el argumento `--template` al comando `create astro`.

Ejecuta el siguiente comando en tu terminal, sustituyendo el nombre de la plantilla oficial de Astro para principiantes, o el nombre de usuario de GitHub y el repositorio del tema que quieras utilizar:

```bash
# crea un nuevo proyecto con un ejemplo oficial
bun create astro@latest --template <example-name>
# crea un nuevo proyecto basado en la rama principal de un repositorio de GitHub
bun create astro@latest --template <github-username>/<github-repo>
```

## Desarrollar y construir

Para ejecutar el servidor de desarrollo, utiliza el siguiente comando:

```bash
bun run dev
```

### Crea y previsualiza tu sitio web

To build your site, use the following command:

```bash
bun run build
```

Cuando haya finalizado la compilación, ejecuta el comando de vista previa correspondiente (p. ej, `bun run preview`) en tu terminal y podrás ver la versión compilada de tu sitio web de forma local en la misma ventana de vista previa del navegador.

## Pruebas

Bun incluye un ejecutor de pruebas rápido, integrado y compatible con Jest a través del [comando `bun test`](https://bun.com/docs/cli/test). También puedes utilizar cualquier otra [herramienta de pruebas para Astro](/es/guides/testing/).
