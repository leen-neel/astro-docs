## ¿Por qué utilizar medios alojados?

Los servicios de alojamiento multimedia ayudan a particulares, equipos y organizaciones a almacenar, gestionar, optimizar y distribuir sus recursos de imagen y vídeo mediante APIs dedicadas desde una ubicación central.

Esta centralización puede resultar útil, sobre todo cuando se utiliza una única fuente de información fiable para los activos entre varias propiedades web o móviles. Esto es importante si formas parte de una organización en la que varios equipos necesitan utilizar los mismos recursos, o si estás integrando el sistema con otros sistemas de contenido, como un PIM (gestor de información de productos), para vincular tus recursos a los productos.

Los servicios de alojamiento de imágenes pueden transformar y optimizar tus imágenes, ofreciendo automáticamente versiones optimizadas a tus visitantes. Estas [imágenes remotas](/es/guides/images/#imágenes-remotas) se pueden utilizar en los componentes integrados de Astro `<Image />` y `<Picture />`, y están disponibles para todos los tipos de archivo de tu proyecto, incluidos Markdown, MDX y los componentes del framework de UI.

Los servicios de alojamiento de vídeo como [Mux](/es/guides/media/mux/) pueden ofrecer una distribución de vídeo bajo demanda y en directo de alto rendimiento, junto con reproductores de vídeo personalizables, lo que aporta importantes ventajas en cuanto a fiabilidad y escalabilidad frente a la gestión de contenido local. Se encargan de la transcodificación, la compresión y la transformación del vídeo para ofrecer una experiencia de usuario fluida. Una plataforma como Mux también puede incluir análisis de datos para ayudarte a comprender la interacción de tus usuarios.

## ¿Qué sistemas de alojamiento multimedia funcionan bien con Astro?

Al igual que cuando se utiliza un CMS, te interesará recurrir a servicios alojados que te permitan recuperar tus recursos e interactuar con ellos a través de una API o un SDK. Algunos servicios pueden incluir, además, componentes nativos de Astro para mostrar tus imágenes o vídeos.

## ¿Puedo utilizar Astro sin un sistema de alojamiento multimedia?

¡Sí! Astro ofrece funciones integradas para [almacenar imágenes](/es/guides/images/#dónde-guardar-las-imágenes), incluida la posibilidad de hacer referencia a imágenes remotas.

Sin embargo, Astro no admite vídeo de forma nativa, por lo que recomendamos elegir un servicio como [Mux](/es/guides/media/mux/) para gestionar las necesidades de optimización y transmisión de contenidos de vídeo.
