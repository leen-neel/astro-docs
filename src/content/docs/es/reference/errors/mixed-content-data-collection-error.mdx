> **MixedContentDataCollectionError**: **COLLECTION_NAME** contains a mix of content and data entries. All entries must be of the same type.

## ¿Qué salió mal?

Una colección de contenido no puede contener una combinación de entradas de contenido y datos. Debes almacenar las entradas en colecciones separadas según su tipo.
