> **NoAdapterInstalledServerIslands**: Cannot use server islands without an adapter. Please install and configure the appropriate server adapter for your final deployment.

## ¿Qué salió mal?

Para utilizar las islas de servidor, existen las mismas restricciones que para el renderizado del lado del servidor, por lo que se necesita un adaptador.
