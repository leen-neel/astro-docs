> `COLLECTION_ENTRY_NAME` failed to parse.

## ¿Qué salió mal?

Las entradas de la colección con `type: 'data'` deben devolver un objeto con formato JSON válido (para las entradas `.json`) o YAML (para las entradas `.yaml`).
