> **IncorrectStrategyForI18n**: The function `FUNCTION_NAME` can only be used when the `i18n.routing.strategy` is set to `"manual"`.

## ¿Qué salió mal?

Algunas funciones de internacionalización sólo están disponibles cuando el enrutamiento de Astro está deshabilitado por la configuración `i18n.routing: "manual"`.
