:::caution[Obsoleto]
Obsoleto desde Astro 4.0. El helper de RSS ya no existe con un error de respaldo.
:::

> **GetStaticPathsRemovedRSSHelper**: The RSS helper has been removed from `getStaticPaths`. Try the new @astrojs/rss package instead.

## ¿Qué salió mal?

`getStaticPaths` ya no expone un helper para generar un RSS feed. Recomendamos migrar a la integración [@astrojs/rss](/es/recipes/rss/#setting-up-astrojsrss) en su lugar.
