> **GetStaticPathsRequired**: `getStaticPaths()` function is required for dynamic routes. Make sure that you `export` a `getStaticPaths()` function from your dynamic route.

## ¿Qué salió mal?

En [modo estático](/es/guides/routing/#modo-estático-ssg), todas las rutas deben determinarse en el momento de la compilación. Como tal, las rutas dinámicas deben `exportar` una función `getStaticPaths` que devuelva las diferentes rutas para generar.
