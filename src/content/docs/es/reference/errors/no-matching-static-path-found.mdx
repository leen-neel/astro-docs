> **NoMatchingStaticPathFound**: A `getStaticPaths()` route pattern was matched, but no matching static path was found for requested path `PATH_NAME`.

## ¿Qué salió mal?

Se encontró una [ruta dinámica](/es/guides/routing/#rutas-dinámicas), pero no se encontró la ruta correspondiente para los parámetros solicitados. Esto a menudo es causado por un error tipográfico en la ruta generada o solicitada.
