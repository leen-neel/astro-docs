> Route returned a `RETURNED_VALUE`. Only a Response can be returned from Astro files.

## ¿Qué salió mal?

Solo las instancias de [Response](https://developer.mozilla.org/es/docs/Web/API/Response) pueden devolverse dentro de los archivos de Astro.

```astro title="pages/login.astro"
---
return new Response(null, {
 status: 404,
 statusText: 'Not found'
});

// Alternativamente, para redireccionamientos, Astro.redirect también devuelve una instancia de Respuesta
return Astro.redirect('/login');
---
```
