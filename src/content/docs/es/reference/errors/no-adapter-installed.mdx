> **NoAdapterInstalled**: Cannot use server-rendered pages without an adapter. Please install and configure the appropriate server adapter for your final deployment.

## ¿Qué salió mal?

Para usar el renderizado en el servidor, se debe instalar un adaptador para que Astro sepa cómo generar la salida adecuada para la plataforma de despliegue específica.
