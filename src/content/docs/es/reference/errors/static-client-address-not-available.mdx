> **StaticClientAddressNotAvailable**: `Astro.clientAddress` is only available on pages that are server-rendered.

## ¿Qué salió mal?

La propiedad `Astro.clientAddress` solo está disponible cuando [el renderizado en el servidor](/es/guides/on-demand-rendering/) está habilitada.

Para obtener la dirección IP del usuario en modo estático, se pueden usar diferentes API como [Ipify](https://www.ipify.org/) en un [script del lado del cliente](/es/guides/client-side-scripts/) o puede ser posible obtener la IP del usuario utilizando una función serverless alojada en tu proveedor de alojamiento.
