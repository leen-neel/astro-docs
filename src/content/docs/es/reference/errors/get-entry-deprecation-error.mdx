> **GetEntryDeprecationError**: The `METHOD` function is deprecated and cannot be used to query the "COLLECTION" collection. Use `getEntry` instead.

## ¿Qué salió mal?

Las funciones `getDataEntryById` y `getEntryBySlug` están obsoletas y no se pueden utilizar con colecciones de capas de contenido. Utiliza la función `getEntry` en su lugar.
