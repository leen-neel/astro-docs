## ¿Qué salió mal?

Astro encontró un error desconocido al cargar tus colecciones de contenido.
Esto pudo haber sido causado por ciertos errores dentro de tu archivo `src/content.config.ts` o por algunos errores internos.

Si puedes reproducir este error de forma consistente, te agradeceriamos que [abras una issue](https://astro.build/issues/).
