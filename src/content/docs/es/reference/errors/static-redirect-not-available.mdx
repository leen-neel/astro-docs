> **StaticRedirectNotAvailable**: Redirects are only available when using `output: 'server'` or `output: 'hybrid'`. Update your Astro config if you need SSR features.

## ¿Qué salió mal?

La función `Astro.redirect` solo está disponible cuando [el renderizado en el servidor](/es/guides/on-demand-rendering/) está habilitada.

Para redirigir en un sitio web estático, se puede usar el [atributo meta de actualización](https://developer.mozilla.org/es/docs/Web/HTML/Reference/Elements/meta). Ciertos hosts también proporcionan redireccionamientos basados en configuración (por ejemplo, [redireccionamientos de Netlify](https://docs.netlify.com/routing/redirects/)).
