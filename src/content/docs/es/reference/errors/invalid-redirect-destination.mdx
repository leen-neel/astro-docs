> **Ejemplo de mensaje de error:**<br/>
InvalidRedirectDestination: The redirect from "/posts" to "/blog" is invalid. The destination "/blog" does not match any existing route in your project.

## ¿Qué salió mal?

El destino de una redirección dinámica debe coincidir con un patrón de ruta existente e incluir
todos los parámetros dinámicos de la ruta de origen.
Este error se produce cuando una redirección con parámetros dinámicos apunta a un destino
que no se corresponde con ninguna página de tu proyecto, o cuando la ruta de destino
tiene menos parámetros dinámicos que la ruta de origen.
