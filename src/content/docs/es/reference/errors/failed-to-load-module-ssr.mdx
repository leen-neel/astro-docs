> **FailedToLoadModuleSSR**: Could not import `IMPORT_NAME`.

## ¿Qué salió mal?

Astro no pudo importar el archivo solicitado. A menudo, esto se debe a que la ruta de importación es incorrecta (ya sea porque el archivo no existe o hay un error ortográficos en la ruta)

Este mensaje también puede aparecer cuando se importa un tipo sin especificar que se trata de una [importación de tipo](/es/guides/typescript/#importaciones-de-tipos).
