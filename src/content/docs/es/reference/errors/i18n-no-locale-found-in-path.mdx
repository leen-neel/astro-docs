> **i18nNoLocaleFoundInPath**: You tried to use an i18n utility on a path that doesn't contain any locale. You can use `pathHasLocale` first to determine if the path has a locale.

## ¿Qué salió mal?

Una utilidad de i18n intentó utilizar el idioma de una URL que no contiene uno. Puedes evitar este error usando pathHasLocale para verificar si las URL contienen un idioma antes de usar utilidades de i18n.
